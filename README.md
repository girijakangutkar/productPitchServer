# ProductPitch - Backend

This repository contains the **backend codebase** for ProductPitch.  
It powers product search, category filtering, and AI chat responses.

---

## Features
- **RESTful API** for product data
- **Search & Filter Endpoints** for products
- **AI Chat Integration** using LLaMA
- **Authentication-ready** (JWT / session-based)
- **Scalable architecture** for production deployment

---

##  Tech Stack
- Node.js
- Express.js
- MongoDB / PostgreSQL (depending on setup)
- LLaMA (AI model integration)
- dotenv (environment management)
- Render / Vercel / AWS (deployment options)

---

##  Installation
```bash
# Clone the repository
git clone https://github.com/girijakangutkar/productPitchServer.git

# Navigate into the project
cd productpitch-backend

# Install dependencies
npm install

# Start development server
npm run dev
